# Snake Web Game

## 1. 프로젝트 개요
본 작품은 웹 브라우저에서 바로 실행 가능한 스네이크(Snake) 게임입니다.  
플레이어는 방향키를 이용해 뱀을 조작하며 먹이를 획득하고, 점수를 높이는 것을 목표로 합니다.  
단순하고 직관적인 조작 방식을 바탕으로 누구나 쉽게 플레이할 수 있으며, 짧은 시간 안에 게임의 핵심 재미를 경험할 수 있도록 설계하였습니다.

본 게임은 별도 설치 없이 웹에서 실행 가능하도록 구현되었으며, 제한된 플레이 시간 안에서도 몰입감과 반복 플레이의 재미를 제공하는 것을 목표로 하였습니다.

---

## 2. 기획 의도
스네이크 게임은 짧은 플레이타임과 단순한 규칙만으로도 높은 몰입감과 긴장감을 제공하는 대표적인 고전 게임입니다.  
본 작품은 이러한 클래식 게임의 핵심 재미를 웹 환경에서 직관적으로 즐길 수 있도록 구현한 예시 게임입니다.

특히 다음과 같은 요소에 중점을 두었습니다.

- 누구나 바로 이해할 수 있는 단순한 규칙
- 짧은 시간 안에 몰입 가능한 플레이 구조
- 점수 획득과 생존 중심의 반복 플레이 경험
- 웹 환경에서 바로 실행 가능한 접근성

---

## 3. 주요 특징
- 웹 브라우저에서 바로 실행 가능한 게임
- 방향키만으로 조작 가능한 직관적인 플레이 방식
- 먹이를 획득할수록 점수가 상승하는 구조
- 충돌 시 게임 종료 및 재시작 가능
- 짧은 플레이타임 안에서도 긴장감과 성취감을 제공

---

## 4. 게임 방법

### 조작 방법
- `↑` : 위로 이동
- `↓` : 아래로 이동
- `←` : 왼쪽으로 이동
- `→` : 오른쪽으로 이동

### 진행 방식
- 플레이어는 뱀을 조작하여 화면에 나타나는 먹이를 획득합니다.
- 먹이를 먹을 때마다 점수가 증가합니다.
- 벽 또는 자기 몸에 충돌하면 게임이 종료됩니다.
- 게임 종료 후 안내에 따라 다시 시작할 수 있습니다.

---

## 5. 게임 목표
제한된 플레이 시간과 충돌 조건 안에서 최대한 높은 점수를 획득하는 것이 목표입니다.  
단순한 규칙 속에서도 플레이어는 이동 경로를 빠르게 판단해야 하며, 점수가 높아질수록 더 높은 집중력과 순발력이 요구됩니다.
